项目介绍 Project Overview

项目通过电商平台、冷链系统、AI 数据分析，帮助砀山梨扩大销售，提高附加值。用智能监测、精准种植、智能物流与数据反馈实现产供销数字化。目标：提升产业效率、提高农户收入、打造区域品牌。

冷链监测与路径优化系统

传感器 + LoRa + 边缘计算 + 预警系统
